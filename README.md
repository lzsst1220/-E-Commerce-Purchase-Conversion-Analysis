# Behavioral Drivers of Online Purchase Conversion in Session-Level E-Commerce Data

## Overview

This project investigates the behavioral drivers of online purchase conversion using the Online Shoppers Purchasing Intention Dataset from the UCI Machine Learning Repository.

The study applies exploratory data analysis, feature engineering, statistical modeling, clustering, and data visualization to understand how browsing behavior relates to purchase outcomes.

## Dataset

This project uses the Online Shoppers Purchasing Intention Dataset from the UCI Machine Learning Repository.

Dataset Link:
https://archive.ics.uci.edu/dataset/468/online+shoppers+purchasing+intention+dataset

## Methods

- Data Cleaning and Quality Audit
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Logistic Regression
- Linear Regression
- Two-Way ANOVA
- K-Means Clustering
- Data Visualization and Storytelling

## Key Findings

- PageValues is the strongest behavioral predictor of conversion.
- New visitors convert at higher per-session rates than returning visitors.
- Traffic source produces substantial variation in conversion rates.
- Autumn sessions convert more efficiently than Spring sessions despite lower traffic volume.
